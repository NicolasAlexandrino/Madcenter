
# 🏗️ Madcenter – Landing Page Comercial

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Landing Page profissional para a loja Madcenter Casa & Construção**, localizada em **José de Freitas – PI**.

O site foi criado como parte da **atividade avaliativa da segunda nota**, com o objetivo de simular um projeto real de desenvolvimento web para um cliente.

A página apresenta informações sobre a empresa, seus produtos, diferenciais, formas de pagamento e meios de contato.

---

# 🎯 Objetivo

Desenvolver uma **landing page moderna, responsiva e funcional** para apresentar a empresa Madcenter na internet, facilitando o contato com clientes e a divulgação dos seus produtos.

---

### 📍 Informações da empresa

* Nome: **Madcenter Casa & Construção**
* Localização: **José de Freitas – PI**

### 🎯 Objetivo do site

* Apresentar a loja
* Mostrar os principais produtos
* Informar formas de pagamento
* Facilitar contato com clientes

### 👥 Público-alvo

* Clientes da região
* Pessoas que estão construindo ou reformando
* Profissionais da construção civil

### ⚙️ Funcionalidades solicitadas

* Seção sobre a empresa
* Catálogo de produtos
* Diferenciais da loja
* Formas de pagamento
* Contato com endereço e mapa
* Botão de **WhatsApp**

---

# 🎨 Layout do Site

O design foi pensado para ser **limpo, moderno e profissional**, utilizando cores relacionadas ao setor de construção.

Principais características:

* Layout moderno
* Design responsivo
* Organização clara das informações
* Botões de contato visíveis
* Interface simples e intuitiva
